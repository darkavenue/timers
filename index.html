  2. Click Create a project → name it anything → skip Analytics → Create
    3. >
  3. In the left sidebar, click Build → Realtime Database
    4. >
  4. Click Create Database → pick any region → choose Start in test mode → Enable
    5. >
  5. Copy the URL shown at the top (looks like https://yourproject-default-rtdb.firebaseio.com) and paste it above
    6. >
  6. Make up a room code (any random string — same on all your devices)
    7. >
  7. On your other devices, enter the same URL and room code
    8. >
